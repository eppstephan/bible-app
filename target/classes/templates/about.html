Die Bibel

Nach der Übersetzung von Martin Luther 1912
Diese Web Applikation wurde mit Hilfe von Java, Spring Boot und Thymeleaf realisiert. Für die Entwicklung der Nutzerschnittstelle wurde HTML, CSS und JavaScript verwendet. Der Bibeltext der Übersetzung von Martin Luther von 1912 ist bezogen von https://ebible.org. Geschrieben wurde diese Applikation von Stephan Epp und zuletzt geändert am 11. November 2023.
